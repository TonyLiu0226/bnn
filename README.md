# Your One Stop Source for All Your MCU News

Welcome to the official digital headquarters of the **Bukkake News Network (BNN)**—the most trusted, most relentless, and arguably the *stickiest* news source in the fine fictional city of Bukkake, MU. 

We deliver hard-hitting journalism right to your face. Whether it's the latest developments from City Hall, traffic jams on the sticky streets of downtown, or breaking news on local infrastructure leaks, BNN covers it all with unmatched coverage.

## Admin Access

Although articles are unlocked and unpaywalled to all users as is expected for a number one rated news source,the BNN admin dashboard is highly classified. Only authorized journalists with an account can access this part of the site to edit and create articles! If you don't want to create an account, I guess you are stuck simply reading the news.


## Running Locally

```bash
npm install && npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to witness the magic.
